# Finance Dashboard

A simple finance dashboard built using HTML, CSS, and JavaScript.

## Live Demo
https://finance-dashboard-4xa6rk8wz-sonikaravs-projects.vercel.app/

## Features
- Summary cards (Balance, Income, Expense)
- Transactions table
- Search functionality
- Role-based UI (Admin / Viewer)
- Charts (Income vs Expense, Category)

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
